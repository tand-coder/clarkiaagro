== Description ==

Sublime Blog is a simple, fast, and easily customizable free WordPress blog theme. This theme is perfect for blogging purposes. Sublime Blog is fully SEO optimized, responsive and looks perfect on any device, including desktops, laptops, mobiles, tablets, etc. Sublime Blog is also fully customizable in terms of menu, footer, slider, and options, making websites look appealing. Everything is so simple and well organized, you can create any type of blog easily. The sublime blog is RTL, and translation ready. Whether you are a travel blogger, photographer, or food blogger Sublime Blog is just the perfect choice for you! Check theme demo at https://demo.sublimetheme.com/sublime-blog/, read the documentation at https://docs.sublimetheme.com/sublime-blog/ and get support at https://sublimetheme.com/support/.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation == 
1. You can Enable Slider from Slider Settings in customizer.
2. You can choose Slidery Type from latest posts or post category.
3. When you choose latest post you can also set the number of posts displayed in the slider.
4. When you choose post category all the posts in that category will displayed in the slider.
