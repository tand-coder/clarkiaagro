== Description ==
Sublime Blogger is a beautiful and simple child theme of Sublime Blog. Sublime Blogger is  aimed at travel, food, and lifestyle bloggers. It is a fully responsive WordPress theme to get a high ranking on search engines like Google. Sublime Blogger displays your posts in a beautiful grid layout. Sublime Blogger also features a sticky header where your Site Identity, Menus and Search form slides from side. Likewise, this child theme is translation ready, RTL compatible, and speed optimized. It has a refined style that will give your website a professional look and feel. Check theme demo at https://demo.sublimetheme.com/sublime-blogger/, read the documentation at https://docs.sublimetheme.com/sublime-blogger/ and get support at https://sublimetheme.com/support/. 


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
